# Objects-and-Array
Stutern assignment on Object and Array

Question 4

1. Using Spread
    examples
    const food = { beef: '游볼', bacon: '游볩' };

const cloneFood = { ...food };

console.log(cloneFood);
// { beef: '游볼', bacon: '游볩' }

2. Using Object.assign
     examples
     const food = { beef: '游볼', bacon: '游볩' };

const cloneFood = Object.assign({}, food);

console.log(cloneFood);
// { beef: '游볼', bacon: '游볩' }
